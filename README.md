## Produktområde Arbeid - Produktsside Dagpenger

Produktsside dagpenger rettet mot arbeidsledige og permitterte på www.nav.no/arbeid/. 
Innholdet kan redigeres på www.nav.no/arbeid/cms.

## Utvikle lokalt

```
npm i
npm run dev
```
